Projects

I have worked on a variety of projects that allowed me to develop technical skills and gain hands-on experience. Some of the key projects include:

  • Conquest of Seven Kingdoms - Developed a video game project that recreates the expansive map from the *Song of Ice and Fire* universe. This involved adapting the map into a customized version of the game State.io using the Unity Game Engine.
  • Blue Duke - Designed and built a model airplane from scratch using 3D printing and sculpting techniques. Integrated basic motors and lights, which were controlled using Arduino software and the ESP32 Dev Module.
  • Personal Portfolio Webpage - Created and published a personal website utilizing HTML, CSS, and JavaScript. The site serves as an online resume, showcasing my portfolio, professional experience, and contact information. The website is hosted on GitHub: paruchurisid.github.io.
  • Catch My Bus (Ventech Solutions) - Secured a $10,000 investment from Ventech Solutions for this project. Conducted research and development before the project was fully acquired by Ventech. My contributions included website design, hardware mockups (RFID, Bluetooth beacons, and scanners), and basic software development for the prototype model.
  • Tag Cloud Generator - Developed a Java-based program using Java.io components that generates a tag cloud, displaying the occurrence of words in an HTML page after taking user input.
  • Gyroscopic Camera Project - Built a fully rotating camera controlled by a joystick using the ESP32 Dev Module and Arduino software. The camera can capture pictures and adjust its position dynamically based on user input.
